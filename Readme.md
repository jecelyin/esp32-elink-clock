使用PlatformIO, Arduino，GxEPD2, ESP32-WROOM-32D-N8，SHT-30，RX8010SJ, ES8311, RDA5807M, 4.2寸墨水屏实现下面的程序和UI设计。
屏幕大小：400x300
==========================================

==========================================
功能以及操作说明：
1、首次使用先进入WIFI设置界面，连接WIFI成功后保存WIFI信息到EEPROM，下次开机自动连接。
2、首页显示当前时间、日期、农历、温湿度、TODO list、实时天气和明日天气预报。
3、闹钟功能：可以设置多个闹钟，支持贪睡功能， 倒计时功能。
4、收音机功能：可以搜索FM电台，保存电台列表，调节音量。
5、音乐功能：支持播放本地存储的音乐文件，支持基本的播放控制（播放、暂停、上一曲、下一曲）。
6、天气功能：通过WIFI获取实时天气信息和预报，显示在首页。
7、设置功能：可以设置时间格式（12小时/24小时）、闹钟铃声、屏幕亮度、WIFI配网等。
8、所有设置和状态信息均保存到EEPROM，断电后重启自动恢复。
9、UI设计简洁美观，操作流畅，适合墨水屏显示特性。
10、时间相关，优先通过NTP服务器同步时间，然后更新本地RX8010SJ，确保时间准确。首页时间从RX8010SJ读取显示。
